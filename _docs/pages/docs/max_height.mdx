## Basic usage

### Setting the maximum height

Set the maximum height of an element using the `max-h-full` or `max-h-screen` utilities.

```html
<div className="h-24 ...">
  <div className="h-48 **max-h-full** ...">
    <!-- ... -->
  </div>
</div>
```
