## Basic usage

### Setting the minimum height

Set the minimum height of an element using the `min-h-0`, `min-h-full`, or `min-h-screen` utilities.

```html
<div className="h-48">
  <div className="h-24 **min-h-full**">
    <!-- ... -->
  </div>
</div>
```
