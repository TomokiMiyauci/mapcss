## Basic usage

### Setting the border color

Control the border color of an element using the `border-{color}` utilities.

<Example>
  <div className="max-w-xs mx-auto space-y-1">
    <label>
      <span className="text-slate-900 dark:text-slate-200 text-sm font-medium">
        Email address
      </span>
      <input
        type="text"
        placeholder="jane@example.com"
        className="font-sans block text-sm leading-5 w-full py-2 px-3 border-2 border-rose-600 text-slate-500 rounded-lg shadow-sm focus:outline-none focus:ring focus:ring-rose-200 focus:border-rose-500 dark:text-slate-400 dark:placeholder:text-slate-600 dark:bg-slate-900 dark:border-rose-500 dark:focus:ring-rose-900 dark:focus:border-rose-600"
      />
    </label>
    <span className="text-rose-600 dark:text-rose-500 text-sm">
      This field is required.
    </span>
  </div>
</Example>

```html
<input
  className="border-2 **border-rose-600** ..."
  dark-className="border-2 **border-rose-500** ..."
/>
```

### Changing the opacity

Control the opacity of an element's border color using the color opacity modifier.

<Example>
  <div className="grid lg:grid-cols-3 gap-4 text-white text-sm text-center font-bold leading-6">
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-indigo-500/100
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 w-16 h-16 border-indigo-500/100 border-4"></div>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-indigo-500/75
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 w-16 h-16 border-indigo-500/75 border-4"></div>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-indigo-500/50
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 w-16 h-16 border-indigo-500/50 border-4"></div>
    </div>
  </div>
</Example>

```html
<div className="border-4 border-indigo-500**/100** ..."></div>
<div className="border-4 border-indigo-500**/75** ..."></div>
<div className="border-4 border-indigo-500**/50** ..."></div>
```

You can use any value defined in your [opacity scale](/docs/opacity), or use arbitrary values if you need to deviate from your design tokens.

```html
<div className="border-4 border-indigo-600**/[.55]** ..."></div>
```

### Individual sides

Use the `border-{side}-{color}` utilities to set the border color for one side of an element.

    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-r-indigo-500
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 ring-1 ring-slate-900/5 w-16 h-16 border-indigo-200 dark:border-indigo-600/20 border-4 border-r-indigo-500 dark:border-r-indigo-500"></div>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-b-indigo-500
      </p>

```html
<div className="border-4 border-indigo-200 **border-l-indigo-500** ..."></div>
```

### Horizontal and vertical sides

Use the `border-{x|y}-{color}` utilities to set the border color on two sides of an element at the same time.

<Example>
  <div className="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-x-indigo-500
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 ring-1 ring-slate-900/5 w-16 h-16 border-indigo-200 dark:border-indigo-600/20 border-4 border-x-indigo-500 dark:border-x-indigo-500"></div>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">
        border-y-indigo-500
      </p>
      <div className="p-4 shadow-sm bg-white dark:bg-slate-900 ring-1 ring-slate-900/5 w-16 h-16 border-indigo-200 dark:border-indigo-600/20 border-4 border-y-indigo-500 dark:border-y-indigo-500"></div>
    </div>
  </div>
</Example>

```html
<div className="border-4 border-indigo-200 **border-x-indigo-500** ..."></div>
<div className="border-4 border-indigo-200 **border-y-indigo-500** ..."></div>
```
