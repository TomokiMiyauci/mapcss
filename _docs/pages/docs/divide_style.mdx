## Basic usage

### Set the divide style

Control the border style between elements using the `divide-{style}` utilities.

<Example p="none">
  <div className="grid grid-cols-1 divide-dashed divide-y font-mono text-sm text-center font-bold leading-6 shadow-lg overflow-hidden max-w-sm mx-auto dark:divide-slate-700">
    <div className="p-4 text-slate-400 bg-white dark:bg-slate-800">01</div>
    <div className="p-4 text-slate-400 bg-white dark:bg-slate-800">02</div>
    <div className="p-4 text-slate-400 bg-white dark:bg-slate-800">03</div>
  </div>
</Example>

```html
<div className="divide-y **divide-dashed**">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```
