## Basic usage

### Sizing implicitly-created grid columns

Use the `auto-cols-{size}` utilities to control the size of implicitly-created grid columns.

```html
<div className="grid grid-flow-col **auto-cols-max**">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```
