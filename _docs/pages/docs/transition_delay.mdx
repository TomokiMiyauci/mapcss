## Basic usage

### Delaying transitions

Use the `delay-{amount}` utilities to control an element's transition-delay.

<Example hint="Hover each button to see the expected behaviour">
  <div className="flex flex-col sm:flex-row gap-8 sm:gap-0 justify-around text-white text-sm font-bold leading-6">
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-150</p>
      <button className="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-150">Button A</button>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-300</p>
      <button className="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-300">Button B</button>
    </div>
    <div className="flex flex-col items-center shrink-0">
      <p className="font-medium text-sm text-slate-500 font-mono text-center mb-3 dark:text-slate-400">delay-700</p>
      <button className="px-4 py-2 font-semibold text-sm bg-blue-500 text-white rounded-md shadow-sm hover:scale-125 ease-in-out duration-300 delay-700">Button C</button>
    </div>
  </div>
</Example>

```html
<button className="transition **delay-150** duration-300 ease-in-out ...">Button A</button>
<button className="transition **delay-300** duration-300 ease-in-out ...">Button B</button>
<button className="transition **delay-700** duration-300 ease-in-out ...">Button C</button>
```
