## Basic usage

### Add margin to a single side

Control the margin on one side of an element using the `m{t|r|b|l}-{size}` utilities.

For example, `mt-6` would add `1.5rem` of margin to the top of an element, `mr-4` would add `1rem` of margin to the right of an element, `mb-8` would add `2rem` of margin to the bottom of an element, and `ml-2` would add `0.5rem` of margin to the left of an element.

<Example p="none">
  <div className="relative font-mono text-white text-sm font-bold leading-6 h-56">
    <div className="absolute top-0 left-1/2 -translate-x-1/2 md:-ml-24">
      <div className="flow-root bg-stripes-purple rounded-b-lg">
        <div className="bg-purple-500 rounded-lg shadow-lg p-4 mt-6">mt-6</div>
      </div>
    </div>
    <div className="absolute right-0 top-1/2 -translate-y-1/2">
      <div className="flow-root bg-stripes-purple rounded-l-lg">
        <div className="flex-none bg-purple-500 rounded-lg shadow-lg p-4 mr-4">
          mr-4
        </div>
      </div>
    </div>
    <div className="absolute bottom-0 left-1/2 -translate-x-1/2 md:ml-24">
      <div className="flow-root bg-stripes-purple rounded-t-lg">
        <div className="bg-purple-500 rounded-lg shadow-lg p-4 mb-8">mb-8</div>
      </div>
    </div>
    <div className="absolute left-0 top-1/2 -translate-y-1/2">
      <div className="flow-root bg-stripes-purple rounded-r-lg">
        <div className="flex-none bg-purple-500 rounded-lg shadow-lg p-4 ml-2">
          ml-2
        </div>
      </div>
    </div>
  </div>
</Example>

```html
<div className="**mt-6** ...">mt-6</div>
<div className="**mr-4** ...">mr-4</div>
<div className="**mb-8** ...">mb-8</div>
<div className="**ml-2** ...">ml-2</div>
```

### Add horizontal margin

Control the horizontal margin of an element using the `mx-{size}` utilities.

<Example>
  <div className="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div className="bg-stripes-indigo rounded-lg">
      <div className="bg-indigo-500 rounded-lg shadow-lg p-4 mx-8">mx-8</div>
    </div>
  </div>
</Example>

```html
<div className="**mx-8** ...">mx-8</div>
```

### Add vertical margin

Control the vertical margin of an element using the `my-{size}` utilities.

<Example>
  <div className="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div className="flow-root bg-stripes-pink rounded-lg">
      <div className="bg-pink-500 rounded-lg shadow-lg p-4 my-8">my-8</div>
    </div>
  </div>
</Example>

```html
<div className="**my-8** ...">my-8</div>
```

### Add margin to all sides

Control the margin on all sides of an element using the `m-{size}` utilities.

<Example>
  <div className="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div className="flow-root bg-stripes-blue rounded-lg">
      <div className="bg-blue-500 rounded-lg shadow-lg p-4 m-8">m-8</div>
    </div>
  </div>
</Example>

```html
<div className="**m-8** ...">m-8</div>
```

### Using negative values

To use a negative margin value, prefix the className name with a dash to convert it to a negative value.

<Example>
  <div className="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div className="flex flex-col items-center">
      <div className="relative w-36 h-16 bg-sky-400/20 border border-sky-700/10 rounded-md overflow-hidden"></div>
      <div className="relative -mt-8 bg-sky-500 rounded-md flex items-center justify-center p-4 shadow-lg">
        -mt-8
      </div>
    </div>
  </div>
</Example>

```html
<div className="w-36 h-16 bg-sky-400 opacity-20 ..."></div>
<div className="**-mt-8** bg-sky-300 ...">-mt-8</div>
```
