## Basic usage

### Setting the ring color

Use the `ring-{color}` utilities to set the color of an [outline ring](/docs/ring-width).

<Example>
  <div className="flex justify-center max-w-md mx-auto gap-4 text-white text-sm text-center font-bold leading-6">
    <button className="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm ring-2 ring-offset-2 ring-offset-slate-50 ring-blue-500 dark:ring-offset-slate-900 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Create account</button>
  </div>
</Example>

```html
<button className="... ring-2 **ring-blue-500**">
  Create account
</button>
```

### Changing the opacity

Control the opacity of an element's background color using the color opacity modifier.

<Example>
  <div className="flex justify-center max-w-md mx-auto gap-4 text-white text-sm text-center font-bold leading-6">
    <button className="px-4 py-2 font-semibold text-sm bg-white text-slate-700 border border-slate-300 rounded-md shadow-sm ring-2 ring-offset-2 ring-offset-slate-50 ring-blue-500/50 dark:ring-offset-slate-900 dark:bg-slate-700 dark:text-slate-200 dark:border-transparent">Create account</button>
  </div>
</Example>

```html
<button className="... ring-2 ring-blue-500**/50**">Subscribe</button>
```

You can use any value defined in your [opacity scale](/docs/opacity), or use arbitrary values if you need to deviate from your design tokens.

```html
<button className="... ring-2 ring-blue-500**/[.55]**">Subscribe</button>
```
